How did we classify the grids? Maps
These urbanised grids are then classified into 5 classes -
  • Class I: Sparse settlements with less road infrastructure, which seem to include non-residential areas like industrial zones and airports, or upcoming residential areas.
  • Class II: Sparse settlements with sufficient road infras- tructure, which also seem to be upcoming residential areas.
  • Class III: Moderately dense settlements with proportion- ately sufficient road infrastructure, which seem to be well developed residential areas.
  • Class IV: Dense settlements with a proportionately suf- ficient road infrastructure, which too seem to be well developed residential areas but more formally planned than Class III areas.
  • Class V: Very dense settlements with inadequate road infrastructure, which seem to be slum resettlement areas.
